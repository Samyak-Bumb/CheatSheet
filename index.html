This Git CheatSheet By Samyak
  1. Basic / Important

    Present Working Directory

    pwd

    List Content

    ls

    Change Directory

    cd
  2. Initializing a Repository 
    git init
  3. Viewing the status

    Full status

    git status

    Short status

    git status -s
  4. Committing the staged files

    Opens the default editor to type a long message

    git commit

    Commits with a one-line message

    git commit -m “Message” .
  5. Staging files

    Stages the current directory and all its content

    git add .

    OR

    git add --a

    Stages a single file

    git add file.js

    Stages with a pattern

    git add *.js
  6. Renaming files 
    git mv file.js file.txt
  7. Removing files 
    git rm file.txt
